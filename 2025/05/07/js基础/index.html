  • 设置返回值的关键字 return 只能返回一个值
  • 想要返回两个或多个，return 后面跟对象或数组
  • return 跳出函数了 后面的语句不会执行

##### 普通正则创建
```js

  • [] 包含方括号中的一个字符
    var res3 = /[abc]/  //包含一位小写字母 a 或 b 或 c
  • [^xxx ] 一位非xxx
  • {} 前面一位字符重复的个数
  • {n} 前面一个字符n个
  • {n,m} 前面一个字符n-m个
  • {n,} 前面一个字符大于等于n个
  • [0-9] 任意一位数字
  • [a-z] 任意一位小写字符
  • [A-Z] 任意一位大写字母
  • [\u4e00-u9fa5] 任意简体汉字
  • \d 任意一位数字 \s 一位空白符(空格) \w 一位数字字母下划线
  • \D 任意一位非数字 \S 一位非空白符(空格) \W 一位非数字字母下划线
  • \n n表示数字，表示跟第几个分组的匹配一模一样的字符
    var re4 = /([0-9])\1{1,}/
console.log(re11.test(666))
  • ^开头 $结尾
  • | 或者 ()分组
  • . 表示任意一位字符 \ 转义符 当要检测的内容时正则的规则语法时。例:\ . 的意思就是包含. 而不是任意字符
  • x(?=y) x字符后面紧跟着是不是y
方法
  • test(值)
  • 检测字符串是否符合正则规则 符合返回 true 否则返回 false
  • exec(值)
  • 返回符合正则规则的项组成的数组和它的下标

14. class 类

  • 构造函数的另一种写法
  • 类名首字母大写
  • class 里面必须写构造器 constructor
  • 类的继承 extends 关键字
  • super 代表父类的构造函数
    class People {
constructor(name) {
this.name = name
}
// 除 constructor 之外都相当于原型 __proto__ 内的
greet() {
console.log(`hi,我是${this.name}`)
}
}
const people = new People('小明')
people.greet()
// 继承 extends
class Student extends People {
constructor(name, banji) {
// super 代表父类的构造函数
super(name)
this.banji = banji
}
greet() {
super.greet()
console.log('我是学生' + this.name)
}
}
const stu = new Student('小吴', '8班')
console.log(stu.name)
stu.greet()
  • 共有的属性可以写在构造器外面
  • 静态方法，挂载到类上，只有类可以访问
  • set edit() 对某个属性设置值的方法，edit是个属性
  • get area() 对某个属性取值的方法，area是个属性
    // 长方形
class Rectangle {
// 共有的属性写在构造器外面
width = 100
constructor(height){
this.height = height
}
// 静态方法 挂载到类上，只有类可以访问
static fun(){
console.log('我是静态方法')
}
// set 对某个属性设置值的方法     edit 是对象的属性
set edit([wid,hei]){
this.width = wid
this.height = hei
}
// get 对某个属性获取值的方法     aear 是对象的属性
get area(){
return width*height
}
}
const rectangle = new Rectangle(200)
Rectangle.fun()   // 类的静态方法，只有类可以访问
console.log(rectangle.area)
rectangle.edit = [10, 20]
